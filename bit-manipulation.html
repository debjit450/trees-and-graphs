Bitwise Operators

Computers store data in binary (0s and 1s). Bitwise operators work directly on these bits.

Basic Operators

  • & (AND): 1 if both are 1. Used to turn OFF bits.
  • | (OR): 1 if either is 1. Used to turn ON bits.
  • ^ (XOR): 1 if different. Used to toggle bits.
  • ~ (NOT): Inverses all bits.
  • << (Left Shift): Multiplies by 2.
  • >> (Right Shift): Divides by 2.

Why learn this?

Bit manipulation allows for extremely fast execution and compact data storage (e.g., storing 8 boolean flags in a single byte).

Essential Tricks

These constant-time O(1) hacks are frequent in competitive programming and system design.

1. Check if Power of 2

Powers of 2 have only one bit set (2=10, 4=100).

bool isPowerOfTwo = (x > 0) && ((x & (x - 1)) == 0);

  • x ^ x = 0 (Self-inverse)
  • x ^ 0 = x (Identity)
