1. Adjacency Matrix

A 2D array where matrix[i][j] = 1 if there's an edge between vertex i and j.

ABCDE
A01100
B10110
C11001
D01001
E00110

2. Adjacency List

Each vertex stores a list of its neighbors. Most common representation for sparse graphs.

A: [B, C] B: [A, C, D] C: [A, B, E] D: [B, E] E: [C, D]

Advantages

Space efficient for sparse graphs O(V+E). Fast to iterate over all neighbors of a vertex.

3. Edge List

Simply a list of all edges. Good for edge-centric algorithms like Kruskal's MST.

edges = [ (A, B), (A, C), (B, C), (B, D), (C, E), (D, E) ]

When to Use Which?

  • Adjacency List: Default choice for most graph algorithms
  • Adjacency Matrix: When you need O(1) edge lookup, or dense graphs
  • Edge List: For edge-centric algorithms like Kruskal's MST
