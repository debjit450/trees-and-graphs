Graph Coloring

Problem

Assign colors to vertices so that no two adjacent vertices share the same color, using minimum colors.

Chromatic Number χ(G): The minimum number of colors needed to properly color graph G.

Bipartite Graphs

Key Insight

A graph is bipartite if and only if it can be 2-colored (χ(G) ≤ 2). Equivalently, it contains no odd-length cycles.

function isBipartite(graph): color = {} for node in graph: if node not colored: if not bfsColor(node): return false return true function bfsColor(start): queue = [start] color[start] = 0 while queue not empty: node = queue.dequeue() for neighbor in adj[node]: if color[neighbor] == color[node]: return false // Odd cycle! if neighbor not colored: color[neighbor] = 1 - color[node] queue.enqueue(neighbor) return true

Time: O(V + E)

Greedy Coloring

Algorithm

For each vertex, assign the smallest color not used by its neighbors.

function greedyColor(graph): color = {} for node in graph: usedColors = set() for neighbor in adj[node]: if neighbor in color: usedColors.add(color[neighbor]) // Find smallest available color c = 0 while c in usedColors: c++ color[node] = c return max(color.values()) + 1

Note: Greedy gives at most Δ+1 colors (Δ = max degree), but may not be optimal.

Applications

  • Scheduling (exams, meetings)
  • Register allocation
  • Map coloring
  • Frequency assignment
  • Sudoku solving
