What is an SCC?

Definition

A maximal set of vertices where every vertex is reachable from every other vertex following directed edges.

Within an SCC, you can travel from any node to any other node. SCCs form the "strongly connected" parts of a directed graph.

Kosaraju's Algorithm

Key Idea

Two DFS passes: first to get finish order, second on reversed graph.

function kosaraju(graph): // Pass 1: Get finish order stack = [] for node in graph: if not visited[node]: dfs1(node, stack) // Pass 2: DFS on reversed graph reversed = reverseGraph(graph) sccs = [] while stack not empty: node = stack.pop() if not visited2[node]: scc = [] dfs2(node, reversed, scc) sccs.add(scc) return sccs

Time: O(V + E)

Tarjan's Algorithm

Key Idea

Single DFS using discovery times and lowlink values to identify SCC roots.

function tarjan(node): disc[node] = low[node] = time++ stack.push(node) onStack[node] = true for neighbor in adj[node]: if disc[neighbor] == -1: tarjan(neighbor) low[node] = min(low[node], low[neighbor]) else if onStack[neighbor]: low[node] = min(low[node], disc[neighbor]) // If node is root of SCC if low[node] == disc[node]: scc = [] while true: w = stack.pop() onStack[w] = false scc.add(w) if w == node: break

Time: O(V + E), single pass!

Applications

  • 2-SAT problem
  • Graph condensation
  • Web page clustering
  • Social network analysis
  • Compiler optimization
