Flow Network

Key Concepts

  • Source (S): Origin of flow
  • Sink (T): Destination of flow
  • Capacity: Max flow on an edge
  • Flow: Current flow on edge

Goal: Find the maximum flow from source to sink without exceeding edge capacities.

Max-Flow Min-Cut

Theorem

The maximum flow equals the minimum cut capacity. The min-cut separates S from T with minimum total edge capacity.

Ford-Fulkerson

function fordFulkerson(graph, s, t): maxFlow = 0 while path = findPath(s, t): // Find bottleneck bottleneck = min capacity on path // Augment flow for edge in path: edge.flow += bottleneck edge.reverse.flow -= bottleneck maxFlow += bottleneck return maxFlow

Time: O(E × max_flow) with DFS

Edmonds-Karp

Improvement

Use BFS to find shortest augmenting path. Guarantees O(VE²) time, independent of max flow value.

Applications

  • Network bandwidth
  • Bipartite matching
  • Transportation
  • Sports elimination
  • Image segmentation
