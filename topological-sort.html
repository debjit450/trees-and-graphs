What is Topological Sort?

Definition

A linear ordering of vertices such that for every directed edge (u, v), vertex u comes before v in the ordering.

Important: Only works on Directed Acyclic Graphs (DAGs). If there's a cycle, no valid ordering exists!

Kahn's Algorithm

Key Idea

Repeatedly remove nodes with in-degree 0 (no dependencies). Uses a Queue.

function kahns(graph): inDegree = count incoming edges queue = nodes with inDegree = 0 result = [] while queue not empty: node = queue.dequeue() result.append(node) for neighbor in graph[node]: inDegree[neighbor]-- if inDegree[neighbor] == 0: queue.enqueue(neighbor) return result if len == V else "Cycle!"

Time: O(V + E)

DFS-based Sort

Key Idea

Process nodes in reverse finish order. Add to result when all descendants are done.

function dfsTopoSort(graph): visited = set() stack = [] for node in graph: if node not in visited: dfs(node, visited, stack) return stack.reverse() function dfs(node, visited, stack): visited.add(node) for neighbor in graph[node]: if neighbor not in visited: dfs(neighbor, visited, stack) stack.push(node) // Add after all neighbors done

Time: O(V + E)

Applications

  • Task scheduling
  • Build systems (make, npm)
  • Course prerequisites
  • Spreadsheet calculation
  • Data serialization
