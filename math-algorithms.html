Sieve of Eratosthenes

The efficient way to find all prime numbers up to a limit N. Instead of testing primality for each number individually (O(N√N)), we iteratively mark the multiples of each prime as composite.

The Algorithm

  1. Create boolean array isPrime[0..N], init true.
  2. Loop i from 2 to √N.
  3. If isPrime[i] is true, mark all multiples of i (starting from i²) as false.

Complexity Analysis

Time Complexity O(N log log N)
Space Complexity O(N)

Euclidean Algorithm (GCD)

The Euclidean algorithm finds the Greatest Common Divisor (GCD) of two numbers efficiently using the principle: GCD(a, b) = GCD(b, a % b).

GCD & LCM Relation

Once you have the GCD, finding the Least Common Multiple (LCM) is trivial:

LCM(a, b) = (a * b) / GCD(a, b)

Recursive Implementation

int gcd(int a, int b) { if (b == 0) return a; return gcd(b, a % b); }
