Definitions

Eulerian Path

A path that visits every edge exactly once.

Eulerian Circuit

An Eulerian path that starts and ends at the same vertex.

Hamiltonian Path

A path that visits every vertex exactly once.

Euler Conditions

  • Circuit: All vertices have even degree
  • Path: Exactly 2 vertices have odd degree
  • Neither: More than 2 odd-degree vertices
  • Required: Graph must be connected
// Hierholzer's Algorithm function findEulerPath(start): stack = [start] path = [] while stack not empty: v = stack.top() if v has unused edges: u = next neighbor with unused edge remove edge (v, u) stack.push(u) else: path.prepend(stack.pop()) return path

Hamiltonian

Finding Hamiltonian paths is NP-complete. We use backtracking.

function hamiltonPath(path, visited): if len(path) == V: return path // Found! v = path.last() for neighbor in adj[v]: if neighbor not in visited: visited.add(neighbor) path.push(neighbor) result = hamiltonPath(path, visited) if result: return result path.pop() // Backtrack visited.remove(neighbor) return null
